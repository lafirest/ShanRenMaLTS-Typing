## 支持的功能

- 切换词库
- 朗读词条和编码（听写模式下只朗读词条）
- 临摹编码（看得见字词和编码，听得见朗读，跟着打）
- 听写编码（看不见编码，听得见朗读，自己打）
- 默写编码（看不见编码，听不见朗读）
- 错题复习
- 数据统计（本地）

## 项目说明

本项目是使用开源项目 [Qwerty-Learner](https://github.com/RealKai42/qwerty-learner) 修改定制而实现的，在此声明和感谢。
